# My dotfiles

## Installation

### Dotfiles

```
# Install homesick
gem install homesick

# Clone repository
homesick clone git@bitbucket.org:jez500/dotfiles.git

# Link dotfiles
homesick symlink dotfiles
```
Also see https://github.com/technicalpickles/homesick
